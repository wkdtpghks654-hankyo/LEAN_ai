# Taste Skills Collection

이 디렉토리는 AI가 프론트엔드 코드를 생성할 때 더 세련되고 현대적인 디자인을 만들도록 가이드하는 스킬 모음입니다.

## 스킬 목록

### 1. taste-skill
메인 디자인 프레임워크로, 레이아웃, 타이포그래피, 색상, 간격, 애니메이션 등 전반적인 시각적 품질을 개선합니다.

**주요 기능:**
- 3가지 설정 다이얼: DESIGN_VARIANCE (8), MOTION_INTENSITY (6), VISUAL_DENSITY (4)
- 기술적 규칙: RSC 안전성, 이모지 금지, 뷰포트 안정성
- 금지된 AI 패턴 제거
- Bento 2.0 패러다임 적용

**사용법:** `@skills/taste-skill/SKILL.md`

### 2. redesign-skill
기존 프로젝트를 업그레이드하고 디자인 문제점을 개선하는 스킬입니다.

**주요 기능:**
- 3단계 워크플로우: 스캔 → 진단 → 수정
- 타이포그래피, 색상, 레이아웃, 상호작용성 등 8가지 감사 카테고리
- 기존 기술 스택 유지하면서 개선

**사용법:** `@skills/redesign-skill/SKILL.md`

### 3. soft-skill
프리미엄 럭셔리 미학을 강조하는 하이엔드 비주얼 디자인 프레임워크입니다.

**주요 기능:**
- $150k+ 에이전시급 디지털 경험 생성
- Apple/Linear급 디자인 언어
- Double-Bezel 아키텍처, 커스텀 cubic-bezier 곡선

**사용법:** `@skills/soft-skill/SKILL.md`

### 4. output-skill
불완전한 코드 생성을 방지하고 완전한 구현을 보장하는 스킬입니다.

**주요 기능:**
- placeholder 주석 금지
- 건너뛴 코드 블록 방지
- 완전한 출력 강제

**사용법:** `@skills/output-skill/SKILL.md`

## 사용 방법

Claude Code에서 이 스킬들을 참조할 때:

```
@skills/taste-skill/SKILL.md를 적용하여 랜딩 페이지를 만들어주세요.
```

또는 여러 스킬을 조합하여 사용:

```
@skills/taste-skill/SKILL.md와 @skills/output-skill/SKILL.md를 적용하여...
```
