# 수상 내역 폴더

포트폴리오 페이지의 "수상경력" 탭에 표시될 수상 내역 이미지를 이 폴더에 추가하세요.

## 이미지 파일명 규칙

수상 연도와 제목을 포함하여 명명하는 것을 권장합니다.

예시:
- `2024-국가인증상.png`
- `2023-우수상.png`
- `2022-혁신상.png`

## 권장 사양

- **형식**: PNG 또는 JPG
- **비율**: 가로형 (4:3 또는 16:9 권장)
- **해상도**: 최소 800 x 600px
- **용량**: 각 이미지 2MB 이하 권장

## 사용 위치

- 포트폴리오 페이지 (`portfolio.html`)
- "수상경력" 탭
- 수상 카드에 이미지로 표시

## 참고

현재 포트폴리오 페이지에는 샘플 수상 내역 3개가 있습니다.
실제 수상 내역 이미지와 정보로 교체하시면 됩니다.
