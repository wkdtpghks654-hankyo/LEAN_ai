# USP 이미지 폴더

LEAN AI가 다른 이유 섹션에 사용될 이미지를 이 폴더에 추가하세요.

## 필요한 이미지 (총 4개)

### 1. 창업부터 엑싯까지 경험한 팀
- 파일명: `startup-exit.png` (또는 .jpg)
- 권장 사이즈: 최소 800px x 560px
- 내용: 팀 사진, 창업 관련 이미지

### 2. 국가에서 인증하는 미친 수상 내역
- 파일명: `awards.png` (또는 .jpg)
- 권장 사이즈: 최소 800px x 560px
- 내용: 상장, 트로피, 수상 관련 이미지

### 3. 운영중인 사업 총 4개
- 파일명: `businesses.png` (또는 .jpg)
- 권장 사이즈: 최소 800px x 560px
- 내용: 학쫑프로, 리베라 등 사업 화면

### 4. SKY 출신 개발팀
- 파일명: `sky-team.png` (또는 .jpg)
- 권장 사이즈: 최소 800px x 560px
- 내용: 팀원 사진, 학교 관련 이미지

## 이미지 권장 사양

- **형식**: PNG 또는 JPG
- **비율**: 가로형 (약 4:3 또는 16:9)
- **해상도**: 최소 800 x 560px (고해상도 권장)
- **용량**: 각 이미지 2MB 이하 권장

## 사용 위치

- 홈페이지 "LEAN AI가 다른 이유" 섹션
- 2x2 그리드 레이아웃
